# Primeira Aula de Python

## Comandos Iniciais

* **print()**

    É o comando que "Imprime" o resultado para o usuário.

* **input()**
    
    Usado para receber valores.
    É necessário estabelecer qual o tipo do dado que vai ser inserido, caso não seja dito, ele irá receber os valores como string.

* **len()**

    Faz a contagem de letras de uma string.


## Variáveis

    Variáveis são objetos que recebem dados e os armzenam, podendo ser reutilizadas durante o código.

    Toda variável tem:

    1.Nome;
    2.Tipo;
    3.Valor.

    Variáveis são case sensitive, ou seja, faz diferença se um nome começar com letra maiúscula e o outro letra minúscula.

    * Por Padrão escrevemos com letras minúsculas.

## Tipos de dados primitivos em Python

* **Int**

    Números Inteiros (Ex: 1,2,3,4)

* **Float** 

    Números Reais (Ex: 1.2, 0.78, 90.84)

* **String**

    Texto (Ex: "Nome", "Fui lavar a louça")

* **Bool**

    Verdadeiro ou falso
